Install Visual Studio: Download and install Visual Studio Community Edition from Visual Studio Downloads.

Create a C/C++ Project: Open Visual Studio, create a C/C++ project, and write your code.

Set Breakpoints: Click in the code margin to set breakpoints.

Debug: Press F5 to start debugging. Use the debugging toolbar to step through code.

Inspect Registers/Memory: Use "Registers" and "Memory" windows for advanced debugging.

Stop Debugging: Press Shift+F5 to exit the debugger.
